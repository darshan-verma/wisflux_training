Introduction

Welcome to my technical documentation page. This project is a simple demonstration of my previous learnings.

First some theory about Web/Websites and their working

  • How web work?

    The web works on a client-server model where a browser (the client) sends a request to a web server, which then responds by serving up web pages, scripts, images, or other resources. This communication happens over the Internet using standardized protocols such as HTTP/HTTPS.

  • What is DNS?

    DNS, or Domain Name System, is like the phonebook of the Internet. It translates human-readable domain names (like example.com) into IP addresses, which computers use to identify and communicate with each other.

  • What is Internet?

    The Internet is a global network of interconnected computers and devices that communicate through standardized protocols. It enables the sharing of information and services, ranging from email and websites to streaming media and online gaming.

  • What is hosting?

    Hosting refers to the service of storing a website's files on a server connected to the Internet. This allows users to access the site via a domain name, making the website publicly available online.

HTML Basics

This section covers basic HTML concepts.

HTML (HyperText Markup Language) is the standard markup language used for creating web pages. It is composed of elements (tags) and attributes that define the structure and content of a webpage.

  • Elements + Attributes: HTML is made up of elements, which are represented by tags such as <p>, <div>, or <span>. Elements can contain content and other nested elements. Attributes provide extra details about these elements, such as IDs, classes, or source paths, and are included within the opening tag to modify element behavior or appearance.
  • Forms + Input Type: HTML forms, defined by the <form> element, are used to collect user data. Inside a form, various <input> types are employed to capture different kinds of information such as text, email, passwords, numbers, and more. Other form elements like <textarea>, <select>, and <button> further help in gathering user input in a structured manner.
  • Imports + META Tags: The <link> and <script> tags are used to import external resources like stylesheets and JavaScript files into an HTML document. META tags, placed in the <head> section, provide essential information about the page, such as the character set, viewport settings for responsive design, and metadata for search engines, which helps control how the document is handled and displayed.
  • Lists & Anchor: HTML supports creating ordered lists (<ol>), unordered lists (<ul>), and definition lists (<dl>), which are useful for organizing items in a structured manner. The <a> (anchor) element is used for creating hyperlinks, enabling navigation to external pages, internal sections, or even email addresses. Anchors are vital for both navigation and establishing relationships among content within a document.
  • Tables: Tables in HTML are created using the <table> element and its associated tags, such as <tr>, <th>, and <td>. They are used to display data in a tabular format, with rows and columns, making it easier for users to read and compare information systematically. Using elements like <thead>, <tbody>, and <tfoot> can also add semantic structure to your table data.

HTML Advanced

This section covers advanced HTML topics that enhance web interactivity and multimedia integration.

  • Events: HTML events allow you to create interactive webpages by responding to user actions like clicks, key presses, and mouse movements. Event handling is typically done using JavaScript, where developers attach event listeners to elements to execute code when an event occurs. For example, clicking a button might trigger a JavaScript function.
  • Web Storage: Web Storage provides a way to store key/value pairs locally within the user's browser. Two primary storage types are used: localStorage for persistent storage across sessions, and sessionStorage for temporary storage that lasts only as long as the browser tab remains open. This is useful for saving user preferences and data without involving cookies.
  • Image & Multimedia: HTML supports embedding images, audio, and video to create rich multimedia experiences. The <img> tag displays images, while the <video> and <audio> tags are used to embed videos and sound files, respectively. These elements often include attributes like src, controls, and alt to provide proper functionality and accessibility.
  • Frame: Although traditional frames using the <frame> tag are now outdated, modern HTML uses the <iframe> element to embed another HTML document within the current page. Iframes are useful for displaying content like maps, advertisements, or external web pages without redirecting the user.
  • Canvas & WebGL: The <canvas> element allows for dynamic, scriptable rendering of 2D shapes and images, often used for animations, charts, and games. WebGL extends this functionality to render interactive 3D graphics directly in the browser, providing a powerful platform for complex visualizations and applications.

CSS Basics

This section explains the fundamental concepts of CSS that are essential for styling web pages.

  • Selectors: CSS selectors are patterns used to choose the elements you want to style. They include basic selectors like type, class, and ID selectors, as well as more complex ones such as descendant, attribute, or pseudo-class selectors.
  • Pseudo Selectors: These selectors, such as :hover, :active, and :first-child, target elements based on their state or position in the document tree, enabling interactive styling.
  • Box Model: The box model outlines how an elementâ€™s content, padding, border, and margin contribute to its total dimensions. Mastering the box model is key to managing spacing and layout effectively.
  • Units & Display: CSS uses various units (e.g., pixels, em, rem, percentages) to define sizes. The display property determines how elements are rendered (block, inline, inline-block, etc.), impacting layout behavior.

CSS Advanced

This section covers advanced CSS topics and techniques that empower modern web design and responsive layouts.

  • Flexbox & Grid: Flexbox provides a one-dimensional layout for aligning elements in rows or columns, while Grid offers a two-dimensional layout system to create complex and responsive page structures.
  • CSS Variables: Also known as custom properties, CSS variables allow you to define reusable values that can be referenced throughout your stylesheet, simplifying theme management and maintenance.
  • Media Queries: Media queries let you apply different styles based on the characteristics of the device or screen (such as width or resolution), ensuring your design adapts gracefully to all devices.
  • Transitions & Transforms: These properties enable you to animate changes to CSS properties, creating smooth visual effects and interactive elements without using JavaScript.
  • Specificity & Inheritance: Understanding CSS specificity helps resolve conflicts when multiple rules apply to an element, and inheritance allows child elements to acquire styles from their parent elements, streamlining your code.
